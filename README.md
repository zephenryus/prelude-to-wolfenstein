Prelude of Wolfenstein
======================

Raycasting engine is from Notch's [Prelude of the Chambered 48-hour game](http://s3.amazonaws.com/ld48/index.html).

Level design is Floor 1 from Wolfenstein 3D.

## Pre-requisite

* Java 6 or higher
* Ant installed

## How to run

*
    1) Launch terminal
*
    2) Go to the root project
*
    3) Type : `ant run`
